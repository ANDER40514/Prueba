Manual De Instalacion

Propósito

Este Software esta diseñado para contabilizar la cantida de registro de usuarios que ingresan a la pagina web de la institucion, donde esta a su vez facilitan el uso del usario a la hora de registro y a su vez para el Programador a la hora de inspeccionar la dataBase.

Alcance

  • Solo se utiliza Localmente
  • Solo 1 usuario Utilizando

Requisito Funcionales

  • Windows 10
  • 1GB de RAM
  • 100 de Disco Duro (Preferiblemente SSD)
  • Lector de Codigo
  • SQL Server 2008
  • Blazor

Requisitos no funcionales

  • Debe de Tener acceso a la Base de Datos
  • No se pueden recuperar los datos borrados por el usuario

Estructura DB

  • Nombre(var char)
  • Fecha de Nacimiento (var int)
  • ID (var char)
